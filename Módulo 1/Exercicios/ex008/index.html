Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semâtica)

Nesta frase, tenho um termo em destaque usando a tag STRONG

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semâtica)

Nesta frase, temos um termo em ênfase usando a tag EM (semâtica)

Texto marcado

Podemos também marca um texto usando a tag MARK.

E no outro parágrafo, temos outro texto marcado no final.

Texto GRANDE e texto pequeno

texto grande (forma) com a tag BIG obsoleta
texto pequeno com a tag SMALL com (semântica)

Texto deletado

Podemos marcar um texto excluído para indicar que ele deve ser lido, mas não considerado utilizando a tag DEL.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, usando a tag INS.

Existe também o sublinhado com a tag U ( não semântica).

Texto sobrescrito

Para inserir coisa do tipo (x2+3 e n0) = x2+3, n0 - com a tag SUP

Texto subscrito

Para coisas do tipo (H2O) = H2O - com a tag SUB
